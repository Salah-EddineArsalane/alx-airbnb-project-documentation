## 1. User Management
- User Registration: Create an account with email and password.
- User Login: Authenticate using JWT.
- Profile Management: Update profile info and password.

## 2. Property Management
- Add New Property: Host can list a property (title, location, price, etc.).
- Edit/Delete Property: Host can update or remove listings.
- View Properties: All users can browse available properties.

## 3. Booking System
- Book a Property: Users can reserve available dates.
- View Bookings: Users and hosts can see their bookings.
- Cancel Booking: Users can cancel upcoming reservations.

## 4. Payment Processing
- Secure Payments: Integration with Stripe or PayPal.
- Transaction History: View completed payments and refunds.

## 5. Reviews and Ratings
- Users can rate and review properties.
- Average ratings displayed on property listings.

## 6. Authentication and Authorization
- JWT for secure access.
- Role-based access control (User, Host, Admin).

## 7. Admin Dashboard
- Manage users, listings, and bookings.
- Review reports and flagged content.
